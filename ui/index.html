Claude Desktop Setup

Add this to your Claude Desktop MCP configuration:

{
  "mcpServers": {
    "memoryloom": {
      "command": "node",
      "args": ["/absolute/path/to/memoryloom/server.js"],
      "env": {
        "MEMORYLOOM_API_KEY": "YOUR_API_KEY_HERE"
      }
    }
  }
}

📍 Config Location: ~/Library/Application Support/Claude/claude_desktop_config.json (macOS)

⚠️ Important: MemoryLoom runs as a local stdio server. First run npm start in the MemoryLoom directory.

Local Setup (Recommended)

For best performance, run MemoryLoom locally:

# Clone and setup
git clone https://github.com/bunnysayzz/memoryloom.git
cd memoryloom
npm install
npm start

# Then use in your editor config:
{
  "mcpServers": {
    "memoryloom": {
      "command": "node",
      "args": ["/absolute/path/to/memoryloom/server.js"]
    }
  }
}

💡 Why Local? MCP works best with local processes for stdio communication.
