Claude Desktop Setup

Add this to your Claude Desktop MCP configuration:

{
  "mcpServers": {
    "memoryloom": {
      "url": "${MCP_URL}",
      "headers": {
        "Authorization": "Bearer ${MEMORYLOOM_API_KEY}"
      }
    }
  }
}

Config Location: ~/Library/Application Support/Claude/claude_desktop_config.json (macOS)

HTTP MCP Setup

    "memoryloom": {
      "url": "${MCP_URL}",
      "headers": {
        "Authorization": "Bearer ${MEMORYLOOM_API_KEY}"
      }
    }
  }
}

HTTP MCP: Connect directly to this server without running anything locally. Works for all MCP-compatible editors.
