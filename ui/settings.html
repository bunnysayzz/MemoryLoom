Claude Desktop Configuration

Add this to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "memoryloom": {
      "url": "${MCP_URL}",
      "headers": {
        "Authorization": "Bearer ${MEMORYLOOM_API_KEY}"
      }
    }
  }
}
