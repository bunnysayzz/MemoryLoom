MCP Tools Reference

These are the tools exposed by MemoryLoom to your IDEs.

add_memory

Store new memory with metadata and importance

{"content": "User prefers TypeScript", "importance": 0.9}

search_memories

Find relevant memories using hybrid search

{"query": "programming preferences", "limit": 5}

get_memories

Retrieve specific memories by ID

{"ids": ["memory-id-1", "memory-id-2"]}

update_memory

Modify existing memory content or metadata

{"id": "memory-id", "content": "Updated content"}

delete_memory

Archive or permanently delete memories

{"id": "memory-id", "hard_delete": false}

upsert_memory

Create or update memory by conflict key

{"content": "New or updated memory"}

consolidate_memories

Merge similar memories into summaries

{"minimum_count": 3, "archive_originals": true}

list_memories

List memories with optional filters

{"filters": {"user": "example"}}
