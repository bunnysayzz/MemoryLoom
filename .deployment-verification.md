## ✅ Environment Variables

All platforms support these environment variables:

### Required
- `MEMORYLOOM_STORAGE_MODE` - Storage backend (json/postgres)
- `MEMORYLOOM_DATA_DIR` - Data directory path
- `MEMORYLOOM_LOG_LEVEL` - Logging level

### Optional
- `MEMORYLOOM_DATA_FILE` - Exact JSON file path
- `MEMORYLOOM_BACKUP_ON_WRITE` - Enable backups
- `MEMORYLOOM_BACKUP_RETENTION` - Backup count
- `MEMORYLOOM_POSTGRES_URL` - Postgres connection
- `MEMORYLOOM_API_KEY` - API authentication
- `MEMORYLOOM_HEALTH_PORT` - Health check port
- `MEMORYLOOM_MAX_TOOL_CALLS_PER_MINUTE` - Rate limiting
- `MEMORYLOOM_WEB_UI` - Enable web UI
